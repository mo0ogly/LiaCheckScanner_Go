LiaCheckScanner Go

Scanner IP extractor and RDAP enrichment tool. Extract, enrich, and analyze internet scanner IP addresses.

Go 1.21+ MIT License CI Passing

Features

🔍

IP Extraction

Parses Netfilter .nft files from the internet-scanners repository to extract IPv4 and IPv6 addresses and CIDR blocks.

🌐

RDAP Enrichment

Enriches extracted IPs with WHOIS data from all five Regional Internet Registries: ARIN, RIPE, APNIC, LACNIC, and AFRINIC.

🖥

GUI Interface

Built-in graphical interface using Fyne with a table view, pagination, and interactive controls for browsing results.

📄

CSV Export

Export enriched results to CSV files for further analysis in spreadsheets or other data processing tools.

Quick Start

Get up and running in under a minute.

# Clone the repository
git clone https://github.com/mo0ogly/LiaCheckScanner_Go.git
cd LiaCheckScanner_Go

# Download dependencies
go mod download

# Build the binary
go build -o build/liacheckscanner ./cmd/liacheckscanner

# Run
./build/liacheckscanner

Or use the Makefile:

make build
make run

Architecture

Clean project structure following Go conventions.

cmd/liacheckscanner/ # Main entry point
internal/
    config/ # Configuration management
    extractor/ # IP extraction and RDAP queries
    gui/ # Fyne GUI components
    logger/ # Structured logging
    models/ # Data structures
config/ # Runtime configuration files
docs/ # GitHub Pages website
